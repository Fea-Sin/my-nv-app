# 泛型

泛型的意义在于函数的重用性，设计原则希望组件不仅能够支持当前的数据类型，同时
也能支持未来的数据类型

语法

`<T>(arg: T):T`，其中`T`为自定义变量

上述泛型，可以保证参数跟返回值时相同类型，`T`是一种特殊的变量，只用于表示类型而不是值
